How to run the application
--------------------------
To run our application, you have to have our server running at the same time.
First run the ClientServer application by running StandardServer.java file 
Next you should run the TAid application using the Android Emulator if the following
the ip in the FullscreenActivity.java file should be changed to your internal ip address. Line 223
